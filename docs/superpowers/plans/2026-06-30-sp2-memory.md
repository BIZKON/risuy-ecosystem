# СП-2-память — Движок долгой памяти team-агента (agent_memory) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development или superpowers:executing-plans. Шаги — чекбоксы (`- [ ]`).

**Goal:** team-агент с включённой «Долгой памятью» суммаризирует длинные диалоги (через PII-маску) → эмбедит (РФ-TEI) → хранит в `agent_memory` → подмешивает релевантные сводки прошлого в следующий ответ.

**Architecture:** Чистые правки кода — **нового прод-DDL НЕ требуется** (`agent_memory` создан в СП-1: tenant_id/agent_id FK cascade, kind, content, embedding vector(768), RLS, гранты panel_rw). Зеркалит СП-2a-RAG: бот (owner, обходит RLS) фильтрует tenant_id ЯВНО; эмбеддер — тот же self-host TEI e5 (768); ретрив/запись подмешиваются апстрим в `user_text` (как KB). Суммаризация идёт через существующий masked-LLM-путь `ai.*` (mask→LLM→unmask), хранение — РФ-кластер, эмбеддинг — РФ-TEI: контент во внешний ИИ уходит только маскированным.

**Tech Stack:** Python/asyncpg, pgvector (hnsw), self-host TEI e5 (768), aiogram. Смоуки — `.venv-smoke`.

**Спека:** `docs/superpowers/specs/2026-06-29-sp2-knowledge-memory-design.md` (§4.C память, §4.E сборка промпта, §4.F 152-ФЗ). Опирается на СП-1 (`da5f54e`, схема `agent_memory`) и СП-2a/2b (паттерн RAG + per-tenant изоляция).

```python
async def memory_insert(tenant_id, agent_id, content: str, embedding, *,
                        kind: str = "summary", metadata: dict | None = None) -> None:
    """Записать сводку в долгую память агента. Бот=owner (обходит RLS) → tenant_id/agent_id явные.
    embedding — list[float] vector(768) (kb.embed_passage). metadata.lead — ключ лида (per-lead изоляция)."""
    import json as _json
    if pool is None or not content.strip():
        return
    vec = "[" + ",".join(f"{x:.6f}" for x in (embedding or [])) + "]" if embedding else None
    async with pool.acquire() as c:
        await c.execute(
            "insert into agent_memory (tenant_id, agent_id, kind, content, embedding, metadata) "
            "values ($1,$2,$3,$4,$5::vector,$6::jsonb)",
            tenant_id, agent_id, kind, content.strip(), vec, _json.dumps(metadata or {}),
        )


async def memory_search(embedding, tenant_id, agent_id, lead_key: str | None = None, *,
                        top_k: int = 3, max_distance: float = 0.55) -> list[str]:
    """Top-k сводок памяти агента по косинусной близости. tenant_id+agent_id фильтруем ЯВНО
    (бот=owner). v1: per-lead — только сводки ЭТОГО лида (metadata->>'lead'=$lead) → нет
    кросс-клиентской утечки контекста. Сбой/нет таблицы → []."""
    if not embedding or pool is None:
        return []
    vec = "[" + ",".join(f"{x:.6f}" for x in embedding) + "]"
    async with pool.acquire() as c:
        rows = await c.fetch(
            """
            select content
              from agent_memory
             where embedding is not null
               and tenant_id = $2 and agent_id = $3
               and ($4::text is null or metadata->>'lead' = $4)
               and (embedding <=> $1::vector) <= $5
             order by embedding <=> $1::vector
             limit $6
            """,
            vec, tenant_id, agent_id, lead_key, max_distance, top_k,
        )
    return [r["content"] for r in rows]
```

```python
_PASSAGE_PREFIX = "passage: "      # обязательный e5-префикс для ХРАНИМОГО документа/сводки


async def embed_passage(text: str) -> list[float] | None:
    """Эмбеддинг ХРАНИМОГО текста (сводка памяти) через TEI с passage-префиксом e5.
    None — если эмбеддер не настроен/недоступен (тогда запись памяти пропускаем)."""
    base = (config.EMBEDDER_URL or "").strip().rstrip("/")
    if not base:
        return None
    url = f"{base}/embed"
    headers = {"content-type": "application/json"}
    if config.EMBEDDER_TOKEN:
        headers["authorization"] = f"Bearer {config.EMBEDDER_TOKEN}"
    payload = {"inputs": [_PASSAGE_PREFIX + text], "normalize": True}
    try:
        async with aiohttp.ClientSession(timeout=_EMBED_TIMEOUT) as session:
            async with session.post(url, json=payload, headers=headers) as resp:
                if resp.status != 200:
                    logger.warning("TEI embed(passage) HTTP %s: %s", resp.status, (await resp.text())[:200])
                    return None
                data = await resp.json()
    except Exception as e:  # noqa: BLE001
        logger.warning("TEI embed(passage) недоступен: %s", e)
        return None
    if isinstance(data, list) and data and isinstance(data[0], list):
        return data[0]
    return None
```

```python
"""СП-2-память: долгая память team-агента. Суммаризация диалога (через masked-LLM) → эмбеддинг
(РФ-TEI passage) → agent_memory; ретрив релевантных сводок прошлого для подмешивания в ответ.
Аддитивно и fail-soft: любой сбой → бот отвечает как раньше (память молча отключается).
v1: per-lead (сводки только этого клиента — нет кросс-клиентской утечки), только kind='summary'."""
import logging

import config
import db
import kb

logger = logging.getLogger(__name__)


async def retrieve(text: str, tenant_id, agent_id, lead_key: str | None) -> str:
    """Блок «контекст прошлых диалогов» для подмешивания (или "")."""
    if not agent_id:
        return ""
    vec = await kb.embed_query(text)
    if not vec:
        return ""
    try:
        hits = await db.memory_search(vec, tenant_id, agent_id, lead_key,
                                      top_k=config.MEMORY_TOP_K, max_distance=config.MEMORY_MAX_DISTANCE)
    except Exception as e:  # noqa: BLE001
        logger.warning("memory_search не удался: %s", e)
        return ""
    if not hits:
        return ""
    body = "\n".join(f"• {h.strip()}" for h in hits)
    return "🧠 Контекст прошлых диалогов с этим клиентом:\n" + body


def _dialog_text(history: list[dict]) -> str:
    """Последние сообщения → текст для суммаризации."""
    out = []
    for h in history or []:
        role = "Клиент" if h.get("role") == "user" else "Агент"
        content = (h.get("content") or "").strip()
        if content:
            out.append(f"{role}: {content}")
    return "\n".join(out)


async def maybe_summarize(*, external_id, tenant_id, cfg: dict, history: list[dict],
                          msg_count: int, lead_key: str | None) -> None:
    """Каждые MEMORY_SUMMARIZE_EVERY входящих — суммаризировать недавний диалог и записать в память.
    Best-effort: вызывается ПОСЛЕ отправки ответа, любой сбой проглатывается (не ломает диалог)."""
    import ai  # ленивый импорт (ai импортит многое)
    agent_id = cfg.get("team_agent_id")
    every = config.MEMORY_SUMMARIZE_EVERY
    if not agent_id or every <= 0 or msg_count <= 0 or msg_count % every != 0:
        return
    dialog = _dialog_text(history)
    if not dialog:
        return
    try:
        summary = await ai.summarize_dialog(dialog, cfg)
        if not summary:
            return
        emb = await kb.embed_passage(summary)
        if not emb:
            return
        await db.memory_insert(tenant_id, agent_id, summary, emb,
                               metadata={"lead": lead_key, "up_to": msg_count})
    except Exception as e:  # noqa: BLE001
        logger.warning("maybe_summarize не удался: %s", e)
```
