# Тарифы «ИИ-Агент Про» — токен-модель с ПРОЗРАЧНОЙ маржой

> Числа-примеры (грунтованы на живой себестоимости 20.07.2026), заменяемы. Входные допущения:
> - **Курс продажи токена: 0,0015 ₽/токен** (= 1500 ₽ за 1 млн).
> - **Себест. LLM-токена (DeepSeek V4 Pro, blended): 0,000352 ₽** (234,9/469,8 ₽/млн, подтверждено Timeweb + код).
> - Чат-ответ ≈ 5000 токенов *(калибровать — Pro Thinking жжёт больше)*; маркетинг-пост ≈ 8000.
> - **DaData проверка/поиск: 7,5 ₽** (по твоим данным). **Голос EL: ~9 ₽/мин** (EL $0,08 + Voximplant телефония 1,99 + коннектор 0,31 + запись 0,16). Номер РФ: 420 ₽/мес (pass-through).
> - Наценки: **LLM ×4,26** (вшита в курс), **DaData ×3**, **голос ×2**.

## Как считается маржа (суть)
Ты продаёшь токен за **0,0015 ₽**, LLM-токен стоит тебе **0,000352 ₽** → на каждом LLM-токене маржа **0,001148 ₽ = 76,5 %**.
Не-LLM действия (DaData, голос) стоят в ₽ → их «цена в токенах» = себест.×наценка÷курс (НЕ break-even), чтобы маржа сохранялась.

## Рычаги маржи (что крутить)
- Хочешь пол маржи выше 50 % на Профи → подними наценку голоса до ×2,5–3 (цена 22–27 ₽/мин).
- Если DaData реально ≈0 ₽ (free-tier) → маржа проверки/поиска взлетает до ~93 % при той же цене 22,5 ₽.
- Если прод переключить на V4 Flash (28 ₽/млн) → маржа чата ~98 %, но ломаются ровные пулы; трогать не надо.
